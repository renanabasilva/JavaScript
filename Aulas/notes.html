            Por Marca

                getElementsByTagName()

                * seleciona mais de um objeto

                Ex.: getElementsByTagName('p')[1]

                nome.innerHTML  => pega o texto c/ formatação
                nome.innerText  => pega o texto s/ formatação

                Ex.:
                var p1 = window.document.getElementsByTagName('p')[0]
                window.document.write('Está escrito assim: ' + p1.innerText)         => Vai replicar o texto do primeiro parágrafo
                p1.style.color = 'blue'



            Por ID

                getElementById()

                * Acessa o elemento com base em seu ID exclusivo

                Ex.: 
                    <div id="msg">Texto</div>       => (html)
                    var d = document.getElementById('msg')
                    d.style.background = 'green'
                    d.innerText = 'Estou aguardando'            => Troca o conteúdo da div para: 'Estou aguardando'
                ou:
                    window.document.getElementById('msg').innerText = 'Estou aguardando'



            Por Nome
                
                getElementsByName()

                * O objeto precisa ter a propriedade Name
                ** Também é possível ter mais de um objeto

                Ex.: 
                    <div name="msg">Texto</div>       => (html)
                    var d = document.getElementsByName('msg')[0]
                    d.style.background = 'green'
                    d.innerText = 'Estou aguardando'            => Troca o conteúdo da div para: 'Estou aguardando'



            Por Classe

                getElementsByClassName()

                * Mais de um objeto

                Ex.: 
                    <div class="my-class">Texto</div>       => (html)
                    var d = document.getElementsByClassName('msg')[0]
                    d.style.background = 'green'
                    d.innerText = 'Estou aguardando'            => Troca o conteúdo da div para: 'Estou aguardando'

                

            Por Seletor

                document.querySelector('my-class') - Retorna a primeira ocorrência de um elemento DOM que corresponda a um seletor CSS específico

                document.querySelectorAll() - Retorna 0

                Ex.: 
                    <div id="msg" class="my-class">Texto</div>       => (html)
                    var d = document.querySelector('div#msg') ou var d = document.querySelector('div.my-class')
                    d.style.background = 'green'

Aula 15: Variáveis Compostas

        

            -> Variáveis SIMPLES só armazenam um valor por vez.
            -> Variáveis COMPOSTAS são capazes de armazenar vários valores em uma mesma estrutura.

            Cada elemento de um array (vetor ou váriável composta) é composto por um valor (conteúdo)  e uma chave (indice) de identificação.
            
            Ex.: var num = [5, 8, 4, 7]

            Para adicionar em um indice específico do vetor:

                num[3] = 6   -> Isso faz com que o valor 7 do exemplo seja substituido pelo 6

            Para acrescentar um valor na ultima posição: (igual .append() em python)

                num.push(7)

                num == [5, 8, 4, 6, 7]

            Para saber quantos elementos possui um array:

                num.length

            Para ordenar os elementos do vetor em ordem alfabética:

                num.sort()

                num.sort(function(a, b){return a - b})      -> Para colocar em ordem crescente numérica.
                num.sort(function(a, b){return b - a})      -> Para colocar em ordem decrescente.

            Para inverter a ordem do array: (primeiro vira ultimo e ultimo vira primeiro)

                num.reverse()
            


            Estrutura for para arrays e objects: 

                for(let pos in num) {
                    bloco
                }

                Onde:   let = var 
                        pos = contador
                        in  = dentro
                        num = array

            Para retornar o indice de um elemento:

                num.indexOf(4)

                * Retorna o indice em que o número 4 se encontra (posição dentro do array). Caso não exista o valor dentro do array, retornará -1.
        

        

        
Aula 16: Funções

        

            • Funções são ações executadas assim que são chamadas ou em decorrência de algum evento.
            • Uma Função pode receber parâmetros e retornar um resultado.

            Ex.: 
                function parimpar(n){           -> n é o parâmetro formal
                    if (n % 2 == 0){
                        return 'Par'
                    } else {
                        return 'Impar'
                    }
                }

                let res parimpar(11)            -> 11 é o parâmetro real



            Parâmetros Opcionais

                function soma(n1=0, n2=0){      -> Quando o valor do parâmetro não for declarado, considerará 0 para executar a função.
                    return n1 + n2
                }

                console.log(soma(2))


            
            É possível colocar uma função dentro de uma variável

                let v = function(x){
                    return x*2
                }
                
                console.log(v(5))  ==> 10

            
            ** Estudar ARROW Function



            Função Recursiva

                function fatorial(n){
                    if (n == 1) {
                        return 1
                    } else {
                        return n*fatorial(n-1)      -> A função chama ela mesma (recursividade).
                    }
                }
                
                console.log(fatorial(5))
