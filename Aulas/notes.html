            Variáveis
            Variáveis podem ser declaradas por:
            var e let
            ex: var nome = texto

            Regras p/ variáveis:
            -> Podem começar com letras, $ ou _ 
            -> Não podem começar com números
            -> É possível utilizar letras e números
            -> É possível utilizar acentos e símbolos
            -> Não podem conter espaços
            -> Não podem ser palavras reservadas
            -> Letras maiúsculas e minúsculas tem diferença

            Tipos primitivos
            - Number (Infinity e NaN)
            - String
            - Boolean
            - Null
            - Undefined
            - Object (Array)
            - Function

            * Number engloba os números reais
            ** Em JS, null é um objeto                
        

        

        
Aula 06: Tratamento de Dados

        

            O sinal de '+' serve tanto para cocatenar quanto para somar. Porém para somar é preciso utilizar números
            
            document.write()         -- Pro texto aparecer no html, mas não é salvo
            Number -> String
            Template String
            `asdasd sdasd ${}` -- Para a variavel ficar dentro do texto DEVE-SE utilizar CRASE e não aspas.
            Transformações:
            s.length         -- Quantos caracteres a string possui
            s.toUpperCase()  -- Fica tudo em maiúscula
            s.toLowerCase()  -- Fica tudo em minúsculas
            String(n)        -- Transforma em String
            n.toString()     -- Transforma em String
            Transformações números:
            Number.parseFloat(num)   -- Transforma uma string p/ tipo número real
            Number.parseInt(num)     -- Transforma uma string p/ tipo número inteiro
            Number(num)              -- Transforma uma string p/ tipo número
            n1.toFixed(2)                                                     -- Fica com duas casas decimais
            n1.replace('.', '.')                                              -- Troca o . por ,
            n1.toLocaleString('pt-BR', {style: 'currency', currency: 'EUR'})  -- Faz conversão para o formato de moeda.

Aula 09 e 10: DOM

        

            DOM = Document Object Model

            => É um conjunto de objetos do navegador que dá acesso aos componentes internos do website.

            Árvore DOM

                                    window
                                ⬋     ↓     ⬊
                           location document history
                                       ↓
                                     html
                                   ⬋     ⬊
                                 head     body
                                 ⬋           ⬊
                          meta tittle       h1, p, div
                                                ↓
                                              strong 

            window.document.write('lorem')                      => Escreve no html (temporário)
            window.document.write(window.document.charset)      => Mostra o set de caracteres
            window.document.write(window.navigator.appName)     => Netscape
            window.document.URL                                 => Mostra a URL


            Cada item da árvore DOM é chamado de elemento

            Métodos de acesso aos elementos:

            
    
                
  • por Marca
    • 
                
  • por ID
    • 
                
  • por Nome
    • 
                
  • por Classe
    • 
                
  • por Seletor
    • 
            



            Por Marca

                getElementsByTagName()

                * seleciona mais de um objeto

                Ex.: getElementsByTagName('p')[1]

                nome.innerHTML  => pega o texto c/ formatação
                nome.innerText  => pega o texto s/ formatação

                Ex.:
                var p1 = window.document.getElementsByTagName('p')[0]
                window.document.write('Está escrito assim: ' + p1.innerText)         => Vai replicar o texto do primeiro parágrafo
                p1.style.color = 'blue'



            Por ID

                getElementById()

                * Acessa o elemento com base em seu ID exclusivo

                Ex.: 
                    <div id="msg">Texto</div>       => (html)
                    var d = document.getElementById('msg')
                    d.style.background = 'green'
                    d.innerText = 'Estou aguardando'            => Troca o conteúdo da div para: 'Estou aguardando'
                ou:
                    window.document.getElementById('msg').innerText = 'Estou aguardando'



            Por Nome
                
                getElementsByName()

                * O objeto precisa ter a propriedade Name
                ** Também é possível ter mais de um objeto

                Ex.: 
                    <div name="msg">Texto</div>       => (html)
                    var d = document.getElementsByName('msg')[0]
                    d.style.background = 'green'
                    d.innerText = 'Estou aguardando'            => Troca o conteúdo da div para: 'Estou aguardando'



            Por Classe

                getElementsByClassName()

                * Mais de um objeto

                Ex.: 
                    <div class="my-class">Texto</div>       => (html)
                    var d = document.getElementsByClassName('msg')[0]
                    d.style.background = 'green'
                    d.innerText = 'Estou aguardando'            => Troca o conteúdo da div para: 'Estou aguardando'

                

            Por Seletor

                document.querySelector('my-class') - Retorna a primeira ocorrência de um elemento DOM que corresponda a um seletor CSS específico

                document.querySelectorAll() - Retorna 0

                Ex.: 
                    <div id="msg" class="my-class">Texto</div>       => (html)
                    var d = document.querySelector('div#msg') ou var d = document.querySelector('div.my-class')
                    d.style.background = 'green'
