            Variáveis
            Variáveis podem ser declaradas por:
            var e let
            ex: var nome = texto

            Regras p/ variáveis:
            -> Podem começar com letras, $ ou _ 
            -> Não podem começar com números
            -> É possível utilizar letras e números
            -> É possível utilizar acentos e símbolos
            -> Não podem conter espaços
            -> Não podem ser palavras reservadas
            -> Letras maiúsculas e minúsculas tem diferença

            Tipos primitivos
            - Number (Infinity e NaN)
            - String
            - Boolean
            - Null
            - Undefined
            - Object (Array)
            - Function

            * Number engloba os números reais

Aula 06: Tratamento de Dados

        

            O sinal de '+' serve tanto para cocatenar quanto para somar. Porém para somar é preciso utilizar números
            
            document.write()         -- Pro texto aparecer no html, mas não é salvo
            Number -> String
            Template String
            `asdasd sdasd ${}` -- Para a variavel ficar dentro do texto DEVE-SE utilizar CRASE e não aspas.
            Transformações:
            s.length         -- Quantos caracteres a string possui
            s.toUpperCase()  -- Fica tudo em maiúscula
            s.toLowerCase()  -- Fica tudo em minúsculas
            String(n)        -- Transforma em String
            n.toString()     -- Transforma em String
            Transformações números:
            Number.parseFloat(num)   -- Transforma uma string p/ tipo número real
            Number.parseInt(num)     -- Transforma uma string p/ tipo número inteiro
            Number(num)              -- Transforma uma string p/ tipo número
            n1.toFixed(2)                                                     -- Fica com duas casas decimais
            n1.replace('.', '.')                                              -- Troca o . por ,
            n1.toLocaleString('pt-BR', {style: 'currency', currency: 'EUR'})  -- Faz conversão para o formato de moeda.
